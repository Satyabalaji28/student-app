# Student Management System

## 📌 Description
This is a backend application built using Spring Boot to manage student records. It provides REST APIs to perform CRUD operations with proper validation, exception handling, and MySQL database integration.

---

## 🚀 Features
- Add new student
- View all students
- Get student by ID
- Delete student
- Input validation using annotations
- Global exception handling
- MySQL database integration
- DTO layer for clean API design

---

## 🧠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven


---

## ⚙️ How to Run
1. Clone the repository  
2. Configure MySQL in `application.properties`  
3. Run the application  
4. Use Postman to test APIs  

---

## 📌 API Endpoints

| Method | Endpoint | Description |
|--------|---------|------------|
| POST | /students | Add student |
| GET | /students | Get all students |
| GET | /students/{id} | Get student by ID |
| DELETE | /students/{id} | Delete student |

---

## 💡 Key Learnings
- Implemented layered architecture
- Used DTO for clean API design
- Applied validation and exception handling
- Integrated Spring Boot with MySQL
